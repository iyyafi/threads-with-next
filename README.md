This is a Reddit Thread view using [Next.js](https://nextjs.org/).

## Getting Started

First, cloen and run:

```bash
npm i
# or
npm install
```

Last, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
